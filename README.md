# Rule-Based AI Chatbot

A simple Rule-Based AI Chatbot developed using Java.

## Features

* Greeting responses (Hi, Hello)
* Help command
* Date and Time response
* Thank You response
* Internship information
* Exit command
* Unknown input handling

## Technologies Used

* Java
* Scanner Class
* If-Else Statements
* Loops

## How to Run

1. Compile the program:
   javac RuleBasedChatbot.java

2. Run the program:
   java RuleBasedChatbot

## Learning Outcomes

* Control Flow
* Conditional Statements
* Loops
* User Input Handling
* Basic Chatbot Logic
